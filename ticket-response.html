Resumo da hierarquia

data (array)
├── data[]
│   ├── hugme_status { id, name }
│   ├── ra_status { id, name }
│   ├── favorable_assessment { id, description }
│   ├── information_source { id, name }
│   ├── source { id, name }
│   ├── company { id, name, active }
│   ├── last_feeling { id, name }
│   ├── feed_type { id, name }
│   ├── moderation { user {}, status, reason, request_date, response_date }
│   ├── ra { replicas_count, source_id, source_name, ... }
│   ├── customer { birthday[], cpf[], ..., city[], state[], ... }
│   ├── user { id, name, assignment_date }
│   ├── account { id, name }
│   ├── rafone { expiration_date, status_id, status_name }
│   ├── extra_data[]
│   ├── id, _id, source_external_id, creation_date, ... (campos diretos)
│   ├── historical[] { user {}, auto {}, id, creation_date, type {} }
│   ├── interactions[] { ..., details[] { ... } }
│   └── ...
└── meta
    ├── page { number, size }
    └── total
